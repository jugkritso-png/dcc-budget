# Product Overview

DCC Budget Manager is a web-based budget management system for organizational budget tracking and approval workflows.

## Core Purpose
Manage, track, and analyze organizational budget allocations and expenses with real-time insights and approval workflows.

## Key Features
- Budget allocation and tracking across categories and sub-activities
- Budget request workflow (draft → pending → approved/rejected)
- Monthly budget planning and forecasting
- Expense recording and reporting
- Analytics dashboard with visualizations
- Role-based access control (admin, approver, user)
- Activity logging and audit trails
- Multi-department support

## User Roles
- **Admin**: Full system access, user management, settings configuration
- **Approver**: Review and approve/reject budget requests
- **User**: Create budget requests, view allocated budgets

## Primary Workflows
1. **Budget Request**: Users create requests with line items → Approvers review → Funds deducted on approval
2. **Budget Planning**: Monthly allocation planning across categories and sub-activities
3. **Expense Tracking**: Record actual expenses against allocated budgets
4. **Reporting**: Generate reports and analytics on budget utilization

## Language
Primary language is Thai (ภาษาไทย) with English support. UI text, documentation, and user-facing content should support both languages.
